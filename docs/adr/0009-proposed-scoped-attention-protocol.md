# Proposed Scoped Attention Protocol

Codex does not currently expose a documented `<system-reminder>` tag equivalent for user-authored short-lived instructions. A future `/ultra solve` change may translate the original ultra-plan reminder into a small invocation-scoped attention protocol inside `solve.md`, but this should be validated before adoption so it improves focus without adding prompt sediment, duplicating existing solve rules, or polluting tickets, solve records, memories, or repository guidance.
