# Engineering Skills

Skills for software project work: codebase exploration, implementation coordination, validation, and issue execution.

Stable skills in this category should have clear trigger wording, a repeatable workflow, and validation coverage where the workflow has side effects.

`$setup-ultra-skills` adds the small Ultra-specific extension to a repository whose Matt tracker contract already exists. It keeps the base tracker contract authoritative while configuring review publication, Claim, frontier, resource-link, and Solve Record operations.

`$solve-records` manages local outcome-aware Attempt receipts: it surfaces candidate merge state and recovery next actions, advances only safe candidate merge/ship/land requests, and keeps cleanup ownership explicit. `/ultra solve` handoff integration remains governed by its own runbook.
