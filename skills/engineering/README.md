# Engineering Skills

Skills for software project work: codebase exploration, implementation coordination, validation, and issue execution.

Stable skills in this category should have clear trigger wording, a repeatable workflow, and validation coverage where the workflow has side effects.

`/ultra solve` creates local solve records only after finished, reviewable candidates exist. Use `$solve-records` to list those candidates, explain their merge state, advance safe merge/ship/land requests, and clean up local solve worktrees or branches.
